# YESID: Your Efficient and Smart Insurance Datastore
## Policy Insurance Chatbot Project - Frontend

### Description

This project aims to provide users with a platform in which they can interact with an intelligent assistant for the Chilean insurance market.
This repository contains project's frontend, which allows the user to interact with the chatbot.


### Technologies

- React: A JavaScript library for building user interfaces.
- Tailwind CSS: A CSS framework for rapid and responsive design.
- React Router DOM: A routing library for handling navigation in a React application.
- React Hook Form: A library for managing forms in React.
- Swiper: A JavaScript library for creating touch sliders and carousels.

### Setup and Installation
**Prequisites** - To run this, you will need Node.js and npm installed on your computer. You can refer to this installation guide: https://docs.npmjs.com/downloading-and-installing-node-js-and-npm

**Setup** -
Before running the project, you will need the following:

1. Ensure you have the .env file at the root of the project with the following environment variable:
- **VITE_API_KEY**: The URL for the API, (for a local environment: "http://127.0.0.1:8000").

**Installation** - Once we have everything ready, you can just run `docker comopose up` and the project will start to build and run on `http://localhost`.
